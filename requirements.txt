# ══════════════════════════════════════════════════════════════════════════════
#
# ── Hunspell English dictionary ───────────────────────────────────────────────
# Used by: 02_clean_text.py (only)
# NOT needed by: parse_and_clean_stream.py (use this for large corpora instead)
#
# ⚠️  IMPORTANT: use the SCOWL en_US-large dictionary (76,959 words), NOT the
#     standard system hunspell-en-us package (~50k words). The smaller package
#     incorrectly rejects many valid cybernetics terms (e.g. autopoiesis,
#     homeostasis, affordance) and causes over-filtering of technical vocabulary.
#     Both AshbyX and NorbertX must use the same dictionary file to ensure
#     reproducibility across machines.
#
#   Linux (Debian/Ubuntu) — SCOWL en_US-large:
#     sudo apt-get install hunspell
#     # Download SCOWL en_US-large .dic and .aff:
#     #   https://github.com/LibreOffice/dictionaries/tree/master/en
#     # OR generate via SCOWL (http://wordlist.aspell.net/):
#     #   make en_US-large.dic  (requires SCOWL source)
#     # Place en_US.dic and en_US.aff in /usr/share/hunspell/ (replacing defaults)
#     # Verify word count: wc -l /usr/share/hunspell/en_US.dic  # expect ~76,959
#
#   macOS (Homebrew) — SCOWL en_US-large:
#     brew install hunspell
#     # Download SCOWL en_US-large .dic and .aff from LibreOffice dictionaries
#     sudo mkdir -p /usr/local/share/hunspell
#     sudo cp en_US.dic en_US.aff /usr/local/share/hunspell/
#     # Verify: wc -l /usr/local/share/hunspell/en_US.dic  # expect ~76,959
#
#   Windows:
#     Download SCOWL en_US-large en_US.dic and en_US.aff from:
#       https://github.com/LibreOffice/dictionaries/tree/master/en
#     Place en_US.dic in the project data/ directory.
#     The script will find it automatically.
#     Verify word count: find /c /v "" data\en_US.dic  # expect ~76,959 lines
#
# ── Python fallback (if Hunspell is unavailable) ──────────────────────────────
# pyspellchecker works on all platforms with no system dependencies.
# Slightly less precise than Hunspell but covers the same vocabulary range.
#
#   pip install pyspellchecker>=0.7
#
# ── Automatic degradation ─────────────────────────────────────────────────────
# If neither Hunspell nor pyspellchecker is available, 02_clean_text.py runs
# without word filtering and prints a clear warning with install instructions.
# TF-IDF min_df=2 in the downstream pipeline handles most residual OCR noise.
